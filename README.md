# Insight

Error Insight for Learning

## 快速开始

### 1. 创建并激活虚拟环境

#### macOS/Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

#### Windows

```bash
python -m venv venv
.\venv\Scripts\activate
```

### 2. 安装依赖项

```bash
pip install -r requirements.txt
```

### 3. 运行应用

```bash
streamlit run app.py
```

### 4. 关闭虚拟环境

```bash
deactivate
```
